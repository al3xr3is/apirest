Este é uma api REST a pedido da empresa b2w como desafio para processo seletivo


Foi feito o backend trabalhado na linguagem PHP e banco de dados mysql sem utilização de framework para fins de conhecimento


Não foi utilizado nenhuma tecnologia assincrona para exibição dos dados por causa de praticidade, porém em ambiente de produção isso será pensado antes do inicio de um projeto ;)
